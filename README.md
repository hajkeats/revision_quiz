# revision_quiz
Quiz yourself on your revision topic using the terminal!

Within a `quiz.json` file, create a structure like:
```json
    {
        "Topic 1": {
            "Question 1": "Answer 1",
            "Questions 2: "Answer 2"
        },
        "Topic 2" : {
            ...
    }
```
An example file exists as `quiz.json.demo`.    
Then run `./quiz.py` and quiz yourself!
